## Descrição do Sistema:

O Sistema de Gerenciamento de Biblioteca foi desenvolvido em Python utilizando Programação Orientada a Objetos (POO) e PostgreSQL. O sistema permite o cadastro de livros, revistas e usuários, além do controle de empréstimos e da persistência dos dados em banco de dados relacional.

## Tecnologias Utilizadas:

- Python
- PostgreSQL
- psycopg2
